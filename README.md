# 🎮 League of Legends Damage Checker Bot

Tento Discord bot využíva **Riot Games API** na sledovanie výkonu hráčov v zápasoch. Jeho hlavnou úlohou je zistiť, či daný hráč udelil najviac poškodenia (damage) championom v rámci celého zápasu (všetkých 10 hráčov).

## 🛠️ Inštalácia

Pred prvým spustením je potrebné nainštalovať Python a požadované knižnice:

```bash
pip install discord.py requests python-dotenv
